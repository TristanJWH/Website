My Terminal

Alacritty

I use alacritty as my terminal emulator because it is fast, even if I can't claim that I notice that it is supossedly the fastest. This is important to me because if I want to use my terminal it is because I won't to do something quickly therfore waiting around for terminal to load is not particually appealing. Inside of alacritty I use the nord colour theme because it is a nice-looking, cool-coloured colour scheme.

ZSH

If I honest I am not sure why I use ZSH. I think I tried it and then never switched back. Now I use it because I don't really care. I have my ZSH config stored in ~/.config/zsh which you may know is different to the default mess of files in my home folder. To do this yourself you add the line export ZDOTDIR=$HOME/.config/zsh to your ~/.zshenv file.

Starship

I use the starship cross-shell prompt because it is more easily customiserable than many other prompts. Also, I like that it is cross-shell because it means that if I ever get a resson to change shells then I can keep my prompt with no changes. I have my prompt set up to shoe me my working path and then a ‚ùØ symbol on the left all using my terminal's cyan colour. I like this simple prompt because it doesn't distract me and due to how little I need to see other information. Then again I may end up adding a git part to my prompt because I hope to get better at uploading things to my git server.
